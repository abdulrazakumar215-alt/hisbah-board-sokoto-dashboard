About the Dashboard

The Hisbah Board Sokoto Dashboard is a digital tool designed to support the operations of Hisbah Board Sokoto. It helps streamline report tracking, staff registration, and administrative tasks in a secure and organized way.

This dashboard was developed by Abdulrazak Umar using HTML, CSS, and JavaScript. It is hosted on GitHub Pages and optimized for both desktop and mobile devices.

The goal is to promote transparency, efficiency, and digital transformation within Hisbah operations, while also encouraging community engagement and accountability.

Future updates will include admin login, report filtering, multilingual support (Hausa/English), and integration with mobile apps.
