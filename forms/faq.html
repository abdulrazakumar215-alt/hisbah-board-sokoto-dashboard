Frequently Asked Questions

What is the purpose of this dashboard?

The dashboard helps Hisbah Board Sokoto manage reports, register staff, and monitor activities in a structured and digital way.

Who can access the dashboard?

The dashboard is publicly viewable, but admin features like login and report management are restricted to authorized personnel.

How do I register as a staff member?

Visit the registration form via the dashboard and fill in your details including passport photo and ID.

Is my data secure?

Yes. The dashboard is hosted on GitHub Pages and follows basic security practices. Future updates will include enhanced login and data protection.

Can I contribute to this project?

Absolutely! Visit the CONTRIBUTING.md file in the repository to learn how to submit issues, pull requests, or help with translations.
